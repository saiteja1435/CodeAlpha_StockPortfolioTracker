# CodeAlpha Stock Portfolio Tracker

A simple console-based Stock Portfolio Tracker developed using Python as part of the CodeAlpha Python Programming Internship.

## Features
- Hardcoded stock price dictionary
- User input for stock names and quantities
- Calculates individual stock investment values
- Displays total investment value
- Saves portfolio summary to a text file
- Console-based application

## Technologies Used
- Python

## How to Run
1. Download the project.
2. Open a terminal in the project folder.
3. Run:
   python stock_portfolio_tracker.py
